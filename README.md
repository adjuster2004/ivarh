# ivarh
Clieckermann + Ивановский архив

Ивановский архив работает достаточно быстро, но поиск дел занимает большое количество времени.
Проще заранее скачать листы и потом их просматривать. Обязательно прочитайте пользовательское соглашение.

Требования.
1. Разрешение экрана 1920x1080
2. Яндекс браузер с масштабом в 80%


Последовательность.
1. Установить clickermann http://crapware.aidf.org/
2. В папку projects в корне программы, а можно в любой папке, создать папку ivarh.
3. Поместить в нее файлы из проекта.
4. Открыть на сайте страницу Главная - Фонд № - Опись - Дело
5. Запустить приложение и выбрать конфигурацию !!_ivanovo.cms
6. Запустить выполение задачи.
7. Появится окно с предложением ввести номер страницы, с которой необходимо начать / продолжить загрузку листов.
8. Если ничего не вводить, то загрузка начнется с первой страницы.
9. Наблюдать, как дело скачивается на компьютер в папку Загрузки.
